05 responsivo

CSS Responsivo e Media Queries

5 tópicos
✓
O que é design responsivo?
Páginas que se adaptam automaticamente a diferentes tamanhos de tela. 3 pilares: grids fluidos, imagens flexíveis e media queries.
conceito
✓
Media Queries ⭐
@media (max-width: 768px) { .container { width: 100%; } }. Aplica estilos condicionais conforme a largura da tela.
cai na provacódigo
✓
Breakpoints comuns
480px (mobile pequeno) · 768px (tablets) · 1024px (laptops) · 1025px+ (desktops com min-width).
memorizar
✓
Mobile First
Escreve CSS base para mobile e usa min-width para telas maiores. Melhor prática porque prioriza o dispositivo mais usado.
boa prática
✓
Imagens responsivas
img { max-width: 100%; height: auto; }. max-width ≠ width: 100% — com max-width a imagem mantém tamanho original em telas grandes.
cai na provacódigo

var / let / const ⭐
